DESCRIPCIÓN DE LA APLICACIÓN:
La nueva aplicación "Mi Lista de Compra" se enfoca principalmente en la gestión de un inventario, a diferencia de su versión original.
Para esta versión se incorporaron diversos cambios, al cambiarse el enfoque a la gestión de un inventario, se eliminaron la "Lista de compra" y los "Lugares de interés". En este inventario se pueden añadir nuevos productos, ver el detalle específico de cada uno, así como también editar su precio y cantidad o eliminarlo.
Por otro lado se implementó la internacionalización de la aplicación por medio del idioma que posee el dispositivo, tanto en español como inglés, se implementó una paleta de colores que puede verse en todas las pantallas, además de sonidos al iniciar la aplicación y un ícono que puede verse desde fuera, cabe destacar que para un mejor orden de las interfaces se hizo uso de menús en la parte superior.
